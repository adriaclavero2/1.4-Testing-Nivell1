This project includes three Java exercises with unit tests using JUnit:
Exercise 1: A class to manage a collection of books in a library with functionalities to add, retrieve, get by position, insert at a specific position, and remove by title. Includes JUnit verifications to ensure correct functionality and alphabetical order.
Exercise 2: A CalculoDni class that calculates the letter corresponding to a DNI number. Includes a parameterized JUnit test to validate the calculation with 10 predefined DNI numbers.
Exercise 3: A class with a method that throws an ArrayIndexOutOfBoundsException. Includes a JUnit test to verify that the exception is correctly thrown.
